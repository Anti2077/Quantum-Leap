# Quantum Leap v1.0.0

Quantum Leap 的首个公开版本，为 macOS 提供通过 SSH 编排远端 `iperf3` 的实时网络测速体验。

![Quantum Leap 标准测试结果](https://raw.githubusercontent.com/Anti2077/Quantum-Leap/v1.0.0/docs/images/quantum-leap-overview.jpg)

## 首发功能

- 标准 TCP 双向测试：8 并发，上传/下载各 10 秒
- 高级 TCP/UDP 单向测试：可配置方向、并发数和测试时长
- 支持持续运行并随时停止
- 实时带宽曲线与上传/下载完成态对比
- 平均速率、峰值、总传输量、RTT、抖动和重传统计
- 常用服务器管理，密码存储于 macOS Keychain
- SSH 主机密钥变化和远端端口复用的显式安全确认
- 浅色、深色和跟随系统外观

## 下载

- `Quantum-Leap_1.0.0_macOS_arm64.dmg`：适用于 Apple Silicon Mac
- `Quantum-Leap_1.0.0_macOS_arm64.dmg.sha256`：SHA-256 校验文件

## 运行要求

- macOS 13 或更高版本
- 本机和远端均安装 `iperf3` 3.17 或更高版本
- 远端服务器可通过 SSH 密码登录

## 已知限制

- 本次只提供 Apple Silicon (`arm64`) 构建。
- 应用尚未使用 Apple Developer ID 签名或公证。首次启动时请在 Finder 中右键应用并选择“打开”，或在“系统设置 → 隐私与安全性”中确认打开。
