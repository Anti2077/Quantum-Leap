# Quantum Leap v1.1.0

## 重点

本版本扩展了远端连接方式，并强化了失败诊断和进程清理。现在既可以让 Quantum Leap 通过 SSH 管理临时 `iperf3` 服务，也可以直接连接 Docker、`systemd` 或其他方式维护的持久服务。

## 变更

- 新增“SSH 自动管理”和“直连已有服务”两种服务模式；直连模式不需要 SSH 凭据，也不会停止远端持久服务。
- SSH 新增私钥认证和可选私钥口令，同时保留密码认证。
- 远端缺少 `iperf3` 时识别 APT、DNF、YUM、APK、Pacman、Zypper 或 Homebrew，显示可复制的安装命令并支持安装后重新检测。
- 对连接拒绝、超时、无路由和不可达服务显示明确提示及目标地址/测速端口。
- 远端清理会核验进程 PID、`iperf3 -s` 参数和测速端口，先发送 TERM，必要时再发送 KILL；Linux 使用 `/proc`，其他 Unix 远端回退到 `ps`。
- 常用服务器现在保存服务模式、认证方式和私钥路径；密码或私钥口令继续由 macOS Keychain 保护。
- 移除不再维护的 SwiftUI 实现，仓库以 Tauri 2 + Rust + React 版本为唯一桌面应用实现。

## 下载与安装

1. 下载 `Quantum-Leap_1.1.0_macOS_arm64.dmg`。
2. 打开 DMG，将 **Quantum Leap** 拖入 **Applications**。
3. 首次启动时，在 Applications 中右键应用并选择“打开”；如果 macOS 仍阻止运行，请前往“系统设置 > 隐私与安全性 > 仍要打开”。

## 兼容性

- macOS 13 或更高版本
- Apple Silicon (`arm64`)
- 本机和远端均需 `iperf3` 3.17 或更高版本
