# Quantum Leap v1.3.1

## 亮点

Quantum Leap 现在提供 macOS、Windows 与 Linux 三个平台的正式下载包。Windows 和 Linux 本机测速内置固定源码版本的 `iperf3` 3.21，无需额外安装即可开始本机到远端的网络性能测试。

## 新功能与改进

- 新增 Windows 10/11 x64 NSIS 安装程序。
- 新增 Linux x64 AppImage 与 DEB 软件包。
- Windows/Linux 安装包内置 `iperf3` 3.21 sidecar，并在构建时检查二进制、依赖文件与第三方许可证是否完整。
- 保存的敏感凭据分别使用 macOS Keychain、Windows Credential Manager 与 Linux Secret Service。
- 本机进程启动、路径解析、可执行文件检查与 `ping` 探测适配三种桌面平台。
- 更新中英文项目首页、平台标识、界面截图与第三方组件声明。

## 下载与安装

- **macOS ARM64：**下载 DMG，打开后将 Quantum Leap 拖入 Applications。
- **Windows x64：**下载并运行 NSIS `setup.exe` 安装程序。
- **Linux x64：**下载 AppImage 并授予执行权限，或安装 DEB 软件包。

Windows 与 Linux 本机测速默认使用内置 `iperf3`。macOS 本机及所有远端设备仍需提供可执行的 `iperf3` 3.12 或更高版本。

## 兼容性与限制

- macOS 13 Ventura 或更高版本，Apple Silicon (`arm64`)
- Windows 10/11 (`x64`)
- 使用 glibc 的 Ubuntu/Debian x64 桌面环境
- SSH 自动管理与双端互测的远端设备需要 POSIX shell；暂不支持 Windows 远端端点
- macOS 使用 ad-hoc 签名且未经过 Apple 公证；Windows 与 Linux 附件未进行商业代码签名
