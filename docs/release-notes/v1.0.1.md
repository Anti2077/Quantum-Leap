# Quantum Leap v1.0.1

## 重点

Quantum Leap 现正式以 **GNU General Public License v3.0 only** 发布。应用功能与 v1.0.0 保持一致，本版本用于明确源码、修改版和分发包的自由软件许可。

## 变更

- 新增完整的 `GPL-3.0-only` 标准许可证文本。
- npm、Cargo 和 Tauri 分发元数据统一声明 GPLv3 与项目仓库地址。
- README 补充使用、修改、商业使用和再分发义务说明。
- 应用包元数据加入版权与许可证信息。

## 下载与安装

1. 下载 `Quantum-Leap_1.0.1_macOS_arm64.dmg`。
2. 打开 DMG，将 **Quantum Leap** 拖入 **Applications**。
3. 首次启动时，在 Applications 中右键应用并选择“打开”；如果 macOS 仍阻止运行，请前往“系统设置 > 隐私与安全性 > 仍要打开”。

## 兼容性

- macOS 13 或更高版本
- Apple Silicon (`arm64`)
- 本机和远端均需 `iperf3` 3.17 或更高版本

## 签名状态

本版本使用 ad-hoc 签名，未经过 Apple 公证。该签名已通过结构校验，但不代表能够通过 Gatekeeper 的普通双击安装检查。
