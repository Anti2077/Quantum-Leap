# Quantum Leap v1.1.1

## 重点

本版本为常用服务器增加可识别备注，并更新 macOS DMG 的视觉布局。新的 DMG 背景图、窗口尺寸和应用/Applications 图标位置会随正式安装包一起发布。

## 变更

- 常用服务器支持最多 48 个字符的可选备注，列表优先显示备注并保留服务器地址；没有备注的旧配置自动按原格式读取。
- DMG 使用 880×520 的 Quantum Leap 专用背景图，包含连接曲线、网格和拖拽提示，并配置了应用与 Applications 的布局位置。
- macOS 应用窗口高度微调为 825 像素，以适配新增的服务器备注编辑控件。
- 本版本继续包含 SSH 私钥认证、直连已有服务、远端安装引导和安全进程清理能力。

## 下载与安装

1. 下载 `Quantum-Leap_1.1.1_macOS_arm64.dmg`。
2. 打开 DMG，将 **Quantum Leap** 拖入 **Applications**。
3. 首次启动时，在 Applications 中右键应用并选择“打开”；如果 macOS 仍阻止运行，请前往“系统设置 > 隐私与安全性 > 仍要打开”。

## 兼容性

- macOS 13 或更高版本
- Apple Silicon (`arm64`)
- 本机和远端均需 `iperf3` 3.17 或更高版本

## 签名状态

本版本使用 ad-hoc 签名，未经过 Apple 公证。该签名已通过结构校验，但不代表能够通过 Gatekeeper 的普通双击安装检查。
