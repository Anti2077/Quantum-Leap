# Quantum Leap v1.2.0

本次更新加入双 SSH 设备互测，让 Quantum Leap 从“Mac 到服务器”的测速工具扩展为可编排两台远端设备直接互测的网络工作台。

## 新功能

- 新增“本机测速”与“双端互测”拓扑切换。
- 双端模式可分别配置设备 A 与设备 B，测速流量直接在两台设备之间传输，不经过 Mac。
- 支持一键交换 A/B 端点，并分别使用密码或 SSH 私钥认证。
- 支持为远端设备填写自定义 `iperf3` 绝对路径。
- 新增 TCP 重传次数展示与高重传诊断提示。
- 连接配置区与测试结果区现在可以拖动调整宽度。

## 兼容性与可靠性

- 自动搜索远端 `PATH`、QNAP `/opt/bin/iperf3` 与常见 Entware 安装位置。
- 远端缺少 `nohup` 时使用兼容启动方式。
- `iperf3` 3.17+ 继续使用 JSON 流实时输出；旧版本自动回退到 `-i 0.5` 文本间隔输出。
- 完善双端客户端、服务端的停止与异常清理，修复取消测试时的竞态问题。
- 保存的敏感凭据整合到单一 macOS Keychain 凭据库，并按需迁移旧版本条目。
- 优化连接、配置与结果区域的交互和视觉层级。

## 下载与安装

- `Quantum-Leap_1.2.0_macOS_arm64.dmg`：适用于 Apple Silicon Mac。
- `Quantum-Leap_1.2.0_macOS_arm64.dmg.sha256`：SHA-256 校验文件。
- SHA-256：`3f956e7436159639ab7f82947426d39431d62b05333f214d17eb0638c9f78a35`

打开 DMG 后，将 Quantum Leap 拖入 Applications。此版本要求 macOS 13 Ventura 或更高版本。

当前构建使用 ad-hoc 签名，尚未经过 Apple Developer ID 公证。首次启动若被 macOS 阻止，请在 Finder 中右键应用并选择“打开”，或在“系统设置 → 隐私与安全性”中确认打开。
