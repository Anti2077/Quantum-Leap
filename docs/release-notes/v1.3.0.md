# Quantum Leap v1.3.0

## 亮点

本次更新重新设计了测速工作台，并加入完整的中英双语、多网卡绑定控制和应用内设置。连接配置、实时测速与结果区域在标准和紧凑窗口中都更易浏览和操作。

## 新功能与改进

- 界面、运行状态、确认提示和后端错误现在完整支持 English 与简体中文，可在设置中即时切换。
- 新增应用设置菜单，可切换跟随系统、浅色/深色外观、语言，并查看版本、许可证和项目主页。
- 新增客户端与服务端绑定 IP：支持本机客户端、双端互测发起端和 SSH 管理服务端分别指定 IPv4 或 IPv6 地址，适合多网卡与多地址设备。
- 重新设计响应式测速工作台，改善连接配置、网络拓扑、实时图表和指标区域在不同窗口尺寸下的布局。
- 确认提示和关于窗口改用具备焦点管理的无障碍对话框组件。
- 粒子动画会在页面不可见、系统启用减少动态效果或完全空闲时停止调度，并限制高分屏渲染倍率以减少不必要的重绘成本。

## 兼容性与可靠性

- 本机 `iperf3` 探测现在会确认候选文件具备可执行权限，避免选择无效文件。
- 绑定地址会在运行前校验，并在旧版 `iperf3` 不支持 `-B` 时给出明确错误。
- 绑定 IP 同时适用于 JSON 流和兼容文本输出模式。
- 常用服务器配置兼容新增的绑定地址字段，旧配置可继续读取。
- 新增前端工作流、响应式布局、动画调度和主题行为测试。

## 下载与安装

1. 下载 `Quantum-Leap_1.3.0_macOS_arm64.dmg`。
2. 打开 DMG，将 Quantum Leap 拖入 Applications。
3. 首次启动时，在 Applications 中右键 Quantum Leap 并选择“打开”。如果仍被阻止，请前往“系统设置 > 隐私与安全性 > 仍要打开”。

此版本使用 ad-hoc 签名，未经过 Apple Developer ID 公证。

## 兼容性

- macOS 13 Ventura 或更高版本
- Apple Silicon (`arm64`)
- 建议使用 `iperf3` 3.12 或更高版本；3.17+ 使用 JSON 流，旧版本自动回退到兼容文本模式
